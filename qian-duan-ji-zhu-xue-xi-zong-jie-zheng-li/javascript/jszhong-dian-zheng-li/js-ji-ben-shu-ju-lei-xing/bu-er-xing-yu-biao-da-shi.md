## 布尔型



```
a = []
a && a[0]  //false   
a && a[0] ? '非空数组处理1' : '空数组处理2'  // 用作空数组的判断和处理

b = a[0] || 'ccc'  //为空设置默认值进行兼容用 （|| 默认值）进行处理
```



## 表达式

```
null ? 'a' : 'b' //b

undefined ? 'a' : 'b' //b


0 ? 'a' : 'b' //b   注意！！！


'a' ? 'a' : 'b' //a
2 ? 'a' : 'b' //a  
[] ? 'a' : 'b' //a  
```
